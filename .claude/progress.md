## 핵심 성능 향상 요인 분석

### 1. 데이터 누수 수정 (가장 큰 개선 요인)

**문제점 발견:**
```
Train에서 recent_3_dx 계산 시 마지막 이벤트(예측 타겟)의 dx 포함
→ Train: recent_3_dx mean = 7.06
→ Test: recent_3_dx mean = 3.97 (마지막 이벤트 end_x가 NaN이라 제외됨)
```

**수정 방법:**
```python
# 수정 전: 마지막 이벤트 포함 (타겟 누수!)
for i in range(min(5, n_events)):
    row = ep_data.iloc[-(i+1)]
    ...

# 수정 후: 마지막 이벤트 제외 (i=1부터 시작)
for i in range(1, min(6, n_events)):
    row = ep_data.iloc[-(i+1)]
    ...
```

**효과:** Train/Test 피처 분포 일관성 확보 → 일반화 성능 대폭 향상

### 2. 상대 좌표 예측 전략

**이전 (절대 좌표 직접 예측):**
- 모델 → end_x, end_y 직접 예측
- 문제: 위치별 분포 학습 어려움, 타겟 분포 불일치

**현재 (상대 좌표 예측 후 변환):**
- 모델 → dx, dy 예측 → end_x = start_x + dx
- 장점: 이동 패턴 학습에 집중, 필드 위치 무관

### 3. Y축 대칭 데이터 증강

- 축구장 좌우 대칭 특성 활용
- `y → (68 - y)` 변환으로 데이터 2배 확보
- 증강 후 target_dy 평균: 0.0000 (완벽 대칭)
